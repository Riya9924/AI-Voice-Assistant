# AI-Based Voice-Controlled Desktop Assistant

## Overview
This project is a voice-controlled desktop assistant developed using Python. It recognizes voice commands and performs basic desktop tasks.

## Features
- Voice Recognition
- Text-to-Speech
- Open Google
- Open YouTube
- Tell Current Time
- Wikipedia Search

## Technologies Used
- Python
- SpeechRecognition
- pyttsx3
- Wikipedia
- PyAudio

## How to Run
1. Install the required packages:
   pip install -r requirements.txt

2. Run the project:
   python main.py
